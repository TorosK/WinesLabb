Vad är en applikationsserver och varför behövs den?
- Its a mixed framework or software that allows both creation of web applications and a server enviorment to run them. 
- Its useful for publicity and branding, Customer service and Competitive edge. 
- It transforms data into dynamic content and enbables the functionality of the application. 

Vilka parametrar skulle du titta på för att bedöma vilket IDE som skall användas i ett utvecklingsprojekt?
- The IDE should have a texteditor, compiler and a debugger. The IDE should be able to integrate with other tools such as for example Maven.
